- Fixed pickup slots
- Airport rides only
- Automatic Google Maps mileage
- Stripe Checkout
- First come, first served language

## Pickup times
8:00 PM, 10:00 PM, 12:00 AM, 2:00 AM, 4:00 AM, 10:00 AM, 12:00 PM

## Render setup
Build command: npm install
Start command: npm start

Environment variables:
STRIPE_SECRET_KEY=your Stripe secret key
GOOGLE_MAPS_API_KEY=your Google Maps API key
DOMAIN=https://HoustonAirportSlots.com

## Edit before launch
Replace +10000000000 in index.html with your real phone number.
Change rates in server.js: BASE_FARE, PER_MILE, MINIMUM_FARE.
